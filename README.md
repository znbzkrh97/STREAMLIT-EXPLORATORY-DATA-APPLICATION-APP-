**-- 📊 Streamlit Exploratory Data Analysis (EDA) Dashboard -- **
A simple and interactive Streamlit application to upload, explore, and visualize your dataset (CSV or Excel). This tool helps you quickly understand your data, check for missing values, duplicates, and generate insightful visualizations.

**🔹 Features**
1. File Upload
   Upload CSV, TXT, or Excel files (.csv, .txt, .xlsx, .xls, .xlsm, .xlsb).

2. Data Preview
   Quickly preview the first few rows of your dataset.

3. Data Overview
   Summary metrics:
     - Total rows and columns
     - Number of missing values
     - Number of duplicate rows

4. Dataset Information
   Detailed data types, memory usage, and column information.

5. Statistical Summary
   View descriptive statistics including mean, median, min, max, count, and unique values for all columns.

6. Interactive Visualization
   Choose from:-
      - Bar Chart
      - Line Chart
      - Scatter Plot
      - Histogram
      - Box Plot
      - Pie Chart

    Options include:-
       - X-axis and Y-axis selection
       - Optional legend/color grouping
       - Aggregation method (Sum, Mean, Count)

**🔹 User Guide**

1. Upload File : Click on the 📁 Upload Your CSV / Excel File uploader to select your dataset. Supported formats: CSV, TXT, XLSX, XLS, XLSM, XLSB.
2. Preview Your Data: Once uploaded, the first few rows of your dataset will appear under Data Preview.

Check Data Overview: The dashboard will display:-
-mTotal rows and columns
- Missing values count
- Duplicate rows count

3. Explore Dataset Info: Click on Dataset Info to see detailed information including data types and memory usage.

4. View Statistical Summary: Descriptive statistics for numeric and categorical columns are available under Statistical Summary.

5. Interactive Visualization
 - Select the chart type, X-axis, Y-axis, and optional legend.
-  Choose an aggregation method if relevant.
-  Charts are interactive and rendered directly in the dashboard.
